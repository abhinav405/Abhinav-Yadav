## About This Repository

This repository contains three web development projects organized across separate branches. Each branch focuses on a different frontend experience: a polished shopping website, an interactive video streaming interface, and an AI chatbot powered by React and the Gemini API.

It is designed as a compact portfolio of practical web development skills, covering layout design, responsive styling, JavaScript interactivity, React components, and API integration.

## Project Branches

| Branch | Project | Tech Stack | Experience |
| --- | --- | --- | --- |
| `e-commerce-site` | E-Commerce Website | HTML, CSS | A clean shopping interface with product-focused layout and responsive styling. |
| `video-streaming-platform` | Video Streaming Platform | HTML, CSS, JavaScript | A media browsing interface with dynamic frontend interactions. |
| `chatbot-react-gemini` | AI Chatbot | React, Gemini API | A conversational AI chatbot built with modern React and Gemini API integration. |

> If your actual branch names are different, use the branch that matches each project description.

## Getting Started

Clone the repository:

```bash
git clone <repository-url>
cd <repository-folder>
```

View available branches:

```bash
git branch -a
```

Switch to a project branch:

```bash
git checkout <branch-name>
```

## Running The Projects

### HTML, CSS, and JavaScript Branches

For the e-commerce site and video streaming platform, open the main HTML file in your browser:

```bash
index.html
```

For a smoother development experience, you can also run the folder with a local development server.

### React Chatbot Branch

Install dependencies:

```bash
npm install
```

Create an environment file:

```bash
.env
```

Add your Gemini API key:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

Start the development server:

```bash
npm run dev
```
